# Biomarker Rules

## Selection

Carry forward the previous quarter's biomarker goals by default. Add critical new out-of-range biomarkers from the latest available labs.

Common carried-forward markers:

- IGF-1
- MCV
- fasting glucose / glucose
- RDW
- albumin
- hsCRP

Critical new out-of-range markers can include:

- testosterone, free testosterone, SHBG, FSH, LH
- haemoglobin, RCC, haematocrit, MCV, RDW
- glucose, HbA1c, insulin
- LDL, ApoB, triglycerides
- hsCRP, homocysteine
- creatinine, eGFR
- ALT, AST, GGT

## Comparison

For each selected marker:

1. Find the latest result.
2. Find the immediately prior result for the same biomarker.
3. Preserve displayed value, unit, reference range, date, and notes.
4. Compare direction against the goal where a goal exists.
5. Mark status as in range, out of range, excellent, mixed, no update, or needs retest.

If the latest lab date is after quarter end but is clearly the follow-up for the quarter, say that explicitly.

## Interpretation

- Do not diagnose.
- Do not infer medical meaning beyond obvious range status and previously stated goals.
- Separate "wins", "concerns", and "follow-ups".
- Mention a biomarker only if it is a priority, out of range, meaningfully changed, or tied to an experiment.

## PhenoAge

Use `PhenoAge History` for biological-age trend if the post includes a PhenoAge image or section.

Report:

- latest test date
- chronological age
- phenotypic age
- difference
- whether it improved or worsened versus the prior row
